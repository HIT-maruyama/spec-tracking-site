# 技術スタック

## アーキテクチャ

**静的Webアプリケーション** - バックエンド依存なしの純粋なクライアントサイド実装

## コア技術

- **HTML5** - アクセシビリティと構造のためのセマンティックマークアップ
- **CSS3** - レスポンシブデザインによるモダンスタイリング
- **Vanilla JavaScript (ES6+)** - フレームワークなし、モダンブラウザAPIの活用
- **LocalStorage API** - クライアントサイドデータ永続化
- **JSON** - ストレージとエクスポートのためのデータフォーマット

## ブラウザ要件

- **最小要件**: ES6サポート、localStorage、Fetch API
- **対象ブラウザ**: Chrome 60+、Firefox 60+、Safari 12+、Edge 79+

## データ管理

- **プライマリストレージ**: 構造化キーによるブラウザlocalStorage
- **バックアップ**: JSONエクスポート機能
- **初期データ**: 初回訪問時に読み込まれる静的JSONファイル
- **ID生成**: `crypto.randomUUID()`を使用したUUID v4（フォールバック付き）

## ファイル構造

```
/
├── index.html              # プロジェクト一覧ページ
├── project-detail.html     # タブ付きプロジェクト詳細ページ

└── data/
```

## 共通コマンド

### 開発
```bash

python -m http.server 8000
# または
npx serve .
```

### デプロイメント
```bash

git push origin main
```

### テスト
```bash
# ブラウザでの手動テスト

```

## 主要実装パターン

- **コンポーネントベースアーキテクチャ**: データ管理、バリデーション、UI制御のためのモジュラーJavaScriptクラス
- **イベント駆動UI**: 関心の分離が明確なDOMイベントハンドラー
- **イミュータブルデータ更新**: localStorage更新時は常に新しいオブジェクトを作成
- **エラー境界**: 包括的なバリデーションとユーザーフレンドリーなエラーメッセージ
- **プログレッシブエンハンスメント**: JavaScript無しでも基本コンテンツ表示が可能
